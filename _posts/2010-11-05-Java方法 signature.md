方法的Signature
　　
　　方法的Signature是由方法的参数和返回值的类型一起构成的，下面是他们的结构：
　　
　
　"(argument-types)return-type"
　　
　　其中Java程式中参数类型和其对应的值如下：
　　
　
　Signature　　               Java中的类型
　　Z　　　　　　　                 boolean
　　B　　　　　　　                 byte
　　C　　　　　　
　                char
　　S　　　　　　　                 short
　　I　　　　　　　                  int
　　J　　　　　　　                  long
　　F　　　　　　
　                 float
　　D　　　　　　　                 double
　　L fully-qualified-class;　　  
fully-qualified-class
      [                                       数组
